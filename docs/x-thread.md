I built MaintShield for the Zama Developer Program Builder Track.

MaintShield is a confidential reward layer for maintenance teams.

The idea is simple:
Public workflow, private rewards.

Managers can create maintenance tasks and assign them only to whitelisted technician wallets.
Technicians submit completed work.
Managers approve after real-world verification.

Then the reward/KPI amount is encrypted using the Zama Protocol.

Public:
- Task status
- Equipment code
- Technician wallet
- Approval status

Private:
- Reward amount
- KPI score
- Bonus points
- Personal performance

Only the assigned technician can decrypt their own reward.
Other wallets cannot see the confidential value.

This can be used for maintenance bonuses, confidential KPI scoring, contractor payments, and private payroll workflows.

MaintShield = public accountability + confidential compensation.
